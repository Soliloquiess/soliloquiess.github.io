**Vagrant**는 VirtualBox, Hyper-V, Docker, VMware, AWS 등 다양한 가상화 플랫폼 위에서 개발 환경을 코드로 정의하고 자동 생성·관리할 수 있는 오픈 소스 소프트웨어다. Ruby로 작성되었으며, 여러 언어 생태계를 지원한다.

> VirtualBox를 직접 켜고 PuTTY로 포트 포워딩을 일일이 설정하는 번거로움을 줄여준다. Docker, Kubernetes와 함께 로컬 개발 환경 구성 도구로 많이 쓰인다.

---

## 빠른 시작 (Ubuntu bionic64)

1. 초기화
   ```
   vagrant init hashicorp/bionic64
   ```

2. 가상 머신 기동
   ```
   vagrant up
   ```

![KakaoTalk_20220408_170523009](/assets/KakaoTalk_20220408_170523009.png)

3. SSH 접속
   ```
   vagrant ssh
   ```

![KakaoTalk_20220408_170523118](/assets/KakaoTalk_20220408_170523118.png)

공식 문서: https://www.vagrantup.com/

## CentOS 8 환경 구성

[app.vagrantup.com](https://app.vagrantup.com/) 에서 원하는 박스를 검색하고 **New**를 눌러 추가한다.
시작 전 기존 환경은 먼저 제거하고 진행한다.

```
vagrant init centos/8
vagrant up
vagrant ssh
```
