#### git 블로그 사진 안 올라갈떄

평화로운 어느날 이 블로그에 사진을 올리려니까 엑박이 뜨면서 아무리 푸쉬해도 안 올라 갔다.

나 같은 경우 항상 atom에서 드래그 앤 드롭으로 사진을 가져다 놓으면 별 필요 없이 md파일에 사진이 들어갔는데 아무리 찾아도 없어서 방법을 찾아봤다.
